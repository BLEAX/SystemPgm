**Makefile**

- 소프트웨어 빌드 자동화를 위한 도구
    - 대상List : 의존성List
    - [tab]      : 명령List

**대상(target)**

- Make명령어가 궁극적으로 생성하는 것
- 명령이 수행되어 나온 결과 파일 지정

**의존성(dependency)**

- 대상과 대상을 생성하는데 필요한 소스

기본 명령어

make: Makefile을 실행 → 프로젝트 빌드

make clean: 생성된 파일 정리

변수와 매크로 활용

$@ : 현재의 목표 파일 이름

$* : 확장자가 없는 현재의 목표 파일 이름

$< : 현재 목표 파일보다 최근에 갱신된 파일
